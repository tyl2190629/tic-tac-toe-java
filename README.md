# tic-tac-toe-java

A simple console-based, two-player Tic-Tac-Toe game written in Java.
Players take turns entering a number (1–9) to place an X or O on the board.

##Features
- 3×3 board with numbered positions
- Input validation
- Win, lose, or draw detection
Option to replay
- Fully console-based
  
##How to Run
You can run it on:
- OnlineGDB
- Replit
- Your local machine using any Java IDE (IntelliJ, Eclipse, VS Code)

##Compile and run TicTacToe.java and Board.java.

##Compile (local machine)
javac TicTacToe.java Board.java

##Run
java TicTacToe
